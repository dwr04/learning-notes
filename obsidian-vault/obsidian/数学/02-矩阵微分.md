> [!note] 本篇定位
> 上承 [[01-矩阵微分]]有了 $df = \operatorname{tr}\!\left(\left(\frac{\partial f}{\partial X}\right)^{\top}dX\right)$ 这个"翻译器",
> 求梯度 = 把 $df$ 用**微分法则**算出来,再用 **trace 性质**整理成 $\operatorname{tr}(\,?^{\top}dX)$ 的形状,$?$ 就是梯度。

## 1. 微分运算法则

| 法则 | 公式 | 注意 |
|---|---|---|
| 线性 | $d(X \pm Y) = dX \pm dY$ | |
| 乘积 | $d(XY) = (dX)\,Y + X\,(dY)$ | **顺序不能换**(矩阵不可交换) |
| 转置 | $d(X^{\top}) = (dX)^{\top}$ | 微分和转置可交换 |
| 迹 | $d(\operatorname{tr} X) = \operatorname{tr}(dX)$ | |
| 常量 | $dA = 0$ | $A$ 不含自变量就当常数 |
| 逆 | $d(X^{-1}) = -X^{-1}(dX)\,X^{-1}$ | 用得到时再记 |
| 哈达玛 | $d(X \odot Y) = (dX)\odot Y + X \odot (dY)$ | 逐元素,激活函数会用 |

> [!warning] 唯一容易错的法则:乘积法则的顺序
> $d(XY)=(dX)Y+X(dY)$,**不是** $Y(dX)+(dX)X$。
> 微分号"挪"到哪个因子,那个因子换成它的微分,其余因子**原地不动、左右关系保持**。
> 矩阵乘法不交换,顺序一旦错,后面全错。

## 2. trace 的性质

| 性质 | 公式 |
|---|---|
| 转置不变 | $\operatorname{tr}(A) = \operatorname{tr}(A^{\top})$ |
| 线性 | $\operatorname{tr}(A+B)=\operatorname{tr}A+\operatorname{tr}B,\quad \operatorname{tr}(cA)=c\operatorname{tr}A$ |
| **循环** | $\operatorname{tr}(AB)=\operatorname{tr}(BA),\quad \operatorname{tr}(ABC)=\operatorname{tr}(BCA)=\operatorname{tr}(CAB)$ |
| Frobenius 内积 | $\operatorname{tr}(A^{\top}B)=\operatorname{tr}(B^{\top}A)=\sum_{i,j}a_{ij}b_{ij}$ |
| 标量自迹 | 标量 $a = \operatorname{tr}(a)$ |
| 哈达玛搬运 | $\operatorname{tr}\!\big(A^{\top}(B\odot C)\big)=\operatorname{tr}\!\big((A\odot B)^{\top}C\big)$ |

> [!tip] 两个最干活的性质
> **循环**让你把 $dX$ 挪到乘积末尾(才能凑出 $\operatorname{tr}(?^{\top}dX)$);
> **转置不变**让你把 $(dX)^{\top}$ 翻成 $dX$。整个求导过程基本就靠这两条搬运。

## 3. 重点一:$d(WX)$ 对 $X$ 和对 $W$ 不一样

乘积法则:
$$ d(WX) = (dW)\,X + W\,(dX) $$
求导哪个变量,另一个当常数、微分为 0,只剩一项 —— 但**剩下的项里,微分号站的边不同**:

$$ \text{对 } X:\quad d(WX) = W\,dX \qquad(dX \text{ 在 } W \text{ 的右边}) $$
$$ \text{对 } W:\quad d(WX) = (dW)\,X \qquad(dW \text{ 在 } X \text{ 的左边}) $$

设已经化到 $df = \operatorname{tr}(G^{\top}\, d(WX))$,看这"左右"如何决定梯度:

**对 $X$**($dX$ 已在末尾,直接读):
$$ df = \operatorname{tr}(G^{\top} W\, dX) = \operatorname{tr}\!\big((W^{\top}G)^{\top} dX\big) \;\Rightarrow\; \frac{\partial f}{\partial X} = W^{\top}G $$

**对 $W$**($dW$ 卡在中间,用**循环**把它转到末尾):
$$ df = \operatorname{tr}(G^{\top}\,dW\,X) \overset{\text{循环}}{=} \operatorname{tr}(X\,G^{\top}\,dW) = \operatorname{tr}\!\big((G X^{\top})^{\top} dW\big) \;\Rightarrow\; \frac{\partial f}{\partial W} = G X^{\top} $$

> [!tip] 一句话记忆
> 对**右因子** $X$ 求导 → 梯度 $= (\text{左因子})^{\top} G = W^{\top}G$。
> 对**左因子** $W$ 求导 → 梯度 $= G\,(\text{右因子})^{\top} = G X^{\top}$。
> 谁是常数,谁就**带转置贴到 $G$ 的另一边**。

## 4. 重点二:$d(WX-Y)^{\top}$,平方误差里拆出的两项

经典最小二乘 $f = \lVert WX - Y \rVert_F^2 = \operatorname{tr}(E^{\top}E)$,记 $E = WX - Y$。

对 $\operatorname{tr}(E^{\top}E)$ 用乘积法则 —— **两个 $E$ 各被微分一次,得两项**:
$$ df = \operatorname{tr}\big(\,\underbrace{(dE)^{\top}E}_{\text{微分打在 } E^{\top}} + \underbrace{E^{\top}dE}_{\text{微分打在 } E}\,\big) $$

第一项用**转置不变**翻过来:
$$ \operatorname{tr}\big((dE)^{\top}E\big) = \operatorname{tr}\big(E^{\top}dE\big) $$
两项相等,合并出系数 2:
$$ \boxed{\,df = 2\,\operatorname{tr}(E^{\top}\,dE)\,} $$

> [!tip] "两种情况"的本质
> 不是真有两种不同结果。是乘积法则让两个 $E$ 各贡献一项,
> 而 trace 的**转置不变性**让这两项恒等 → 永远合并成系数 2。
> 这就是平方损失梯度里那个 2 的来历。

再把 $dE = d(WX) $ 按**重点一**代入,$E$ 当作那里的 $G$:

$$ \frac{\partial f}{\partial X} = 2\,W^{\top}E = 2\,W^{\top}(WX - Y) $$
$$ \frac{\partial f}{\partial W} = 2\,E X^{\top} = 2\,(WX - Y)\,X^{\top} $$

> [!note] 去向
> 这俩就是线性回归 / 全连接层权重的梯度。之后过 Karpathy 手推 backprop,
> 一层 $Y = WX$ 配平方损失,反传公式跟这里**一字不差**。→ [[反向传播]]
